# 地下城与新娘-汉化MOD

基于贴吧面条易凉大佬的汉化MODv1.44，游戏1.54补丁，统合MODv1.53制作的汉化MOD

仅提供汉化MOD的下载，仅供娱乐。

使用：将汉化补丁放至游戏根目录，并确保各patchX.xp3中汉化MOD的序号X最大  
在官方下载游戏更新补丁，在wiki中查找有关统合MOD的信息
